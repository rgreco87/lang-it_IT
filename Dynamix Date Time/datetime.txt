Current date and time=Data e ora corrente
Hide map=Nascondi mappa
New date and time=Nuova data e ora
NTP server=Server NTP
Show map=Mostra mappa
Time zone=Fuso orario
Use NTP=Usa NTP

:timezone_plug:
> Seleziona il tuo fuso orario dal menu a discesa.
:end

:datetime_map_plug:
> La mappa mondiale evidenzia il fuso orario attualmente selezionato.
>
> Usa il mouse per spostarti sulla mappa e clicca sul paese per il quale vuoi impostare il fuso orario. La zona corrispondente viene selezionata automaticamente nel menu a discesa.
:end

:use_ntp_plug:
> Seleziona 'Sì' per usare il Network Time Protocol (NTP) e mantenere accurato l’orario del server.
> È fortemente raccomandato l’uso di un Server NTP, specialmente se prevedi di utilizzare Active Directory.
:end

:ntp_server1_plug:
> Questo è il Server NTP primario da usare. Inserisci un FQDN (Fully Qualified Domain Name) oppure un indirizzo IP.
:end

:ntp_server2_plug:
> Questo è il Server NTP (2) alternativo da utilizzare se il Server NTP 1 non è disponibile.
:end

:ntp_server3_plug:
> Questo è il Server NTP (3) alternativo da utilizzare se entrambi i Server NTP 1 e 2 non sono disponibili.
:end

:ntp_server4_plug:
> Questo è il Server NTP (4) alternativo da utilizzare se i Server NTP 1, 2 e 3 non sono disponibili.
:end

:current_time_plug:
> Inserisci la data e l’ora correnti. Usa il formato AAAA-MM-GG HH:MM:SS. Il campo è disabilitato quando si utilizza un server NTP.
:end
